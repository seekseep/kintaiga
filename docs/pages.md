# 画面一覧

## 権限体系

### システム権限（`user.role`）

| 権限 | 説明 |
|--------|------|
| admin | システム管理者。全組織の権限をバイパスする |
| general | 一般ユーザー |

### 組織権限（`organizationRole`）

| 権限 | 説明 |
|--------|------|
| owner | オーナー。組織の全権限を持つ |
| manager | マネージャー。メンバー管理・プロジェクト管理が可能 |
| member | メンバー。自分に関する操作のみ可能 |
